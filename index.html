Política de Privacidade

Última atualização: 02 de abril de 2025

O Divisor de Plantão é um aplicativo desenvolvido para auxiliar na divisão e organização de escalas de plantão. Esta política descreve como tratamos as informações dos usuários.

1. Dados coletados

O aplicativo não coleta dados pessoais. Não realizamos cadastro de usuários, não solicitamos nome, e-mail, telefone ou qualquer outra informação pessoal identificável.

2. Armazenamento de dados

Todas as informações inseridas no aplicativo (escalas, plantões, configurações) são armazenadas localmente no dispositivo do usuário ou no servidor de uso próprio configurado pelo usuário. Nenhuma informação é transmitida para servidores de terceiros.

3. Dados de uso e analytics

O aplicativo não utiliza ferramentas de analytics, rastreamento de comportamento ou coleta de dados de uso.

4. Compartilhamento de dados

Não compartilhamos nenhum dado com terceiros, pois nenhum dado é coletado.

5. Permissões do dispositivo

O aplicativo pode solicitar acesso à internet exclusivamente para comunicação com o servidor configurado pelo próprio usuário.

6. Alterações nesta política

Qualquer alteração nesta política será publicada nesta página com a data de atualização.
